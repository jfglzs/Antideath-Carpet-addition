## ACA的所有规则

### 挖掘无减速(noMiningSlowDown)

挖掘方块无减速(当玩家处于 流体/蜘蛛网/漂浮状态/水中 时)

- 类型：`布尔值`
- 默认值：`false`
- 参考选项：`true`，`false`
- 分类：`ACA`，`SURVIVAL`
